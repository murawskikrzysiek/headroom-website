Auris
SPL meter for monitoring discipline and hearing dose tracking
Version 1.2 May 2026 Latest
  • New Target-level calibration wizard — pink-noise guided flow that pins your monitor controller to a chosen dBA reference (e.g. 79 for K-14, 83 for K-12, 85 for cinema). Plays through your selected output, waits for the reading to settle inside a configurable tolerance band, then captures the position with a knob-position note. Calibrations are stored per output device, so monitors and headphones each carry their own physical-position record. A stale-calibration hint nudges you to recalibrate after 90 days.
  • New Monitoring Discipline — tracks how consistently you actually monitor at your chosen reference. New Reference tab in Settings with industry presets (K-20 / K-14 / K-12 / cinema) plus custom target and tolerance band. Live on-target % chip in the menu bar with a green / yellow / orange ladder, three-segment under / on / over bar, and a drift hint. Today's Summary aggregates the same metric across all sessions; per-session figures land in History alongside the dose stats. Useful at any SPL — fills the gap left by NIOSH dose, which stays at 0 % for users who never reach 80 dBA.
  • New Monitoring alerts — opt-in notifications when your level drifts (rolling 30-min Leq vs session baseline) or sustains a deviation off target. Each category has its own 30-min snooze, so silencing a drift alert doesn't muffle a dose alert.
  • New Pair with Lyra — after a target-level calibration, click "Send to Lyra…" to write the position straight into one of Lyra's three reference slots. Pick the slot (1, 2, or 3), confirm the name, and your calibrated listening level is one keystroke away — Ctrl+F10 / F11 / F12 in Lyra. Auris ships the live Apollo monitor level along with the measured dB SPL value, so the slot displays as e.g. "Mix — 79 dB SPL" in Lyra's menu and HUD. Requires Lyra 1.4 or later.
  • Impr Audio tab restructure — settings now follow a three-step calibration journey, with each step explaining what it does and what it depends on. First-time setup actually has a path through the screen instead of a flat list of controls.
  • Impr Silent capture — pink noise now stops the moment you press "Capture position", so you can name the calibration in silence. Pressing "Adjust again" restarts the noise on the same output device.
  • Fix C-weighting filter — corrected a bug in the C-weighting filter implementation. dBC peak detection, the dBA-vs-dBC gap indicator, and conservative dose mode now report correct values.
  • Fix Channel names — driver-supplied channel names now show up correctly on a wider range of audio interfaces.
Version 1.1 April 2026
  • New Input channel selection — pick any individual channel from your audio interface, not just the default stereo pair. Useful for talkback mics, dedicated measurement inputs, or any channel on multi-channel interfaces. Channel names are read directly from the driver (e.g. "TALKBACK 1", "MIC/LINE 2").
  • New Channel names from CoreAudio — the channel picker shows the actual names reported by your audio driver instead of generic "Channel 1", "Channel 2" labels.
  • Fix Audio engine now correctly re-applies channel routing after a hardware configuration change (e.g. sample-rate switch or device reconnect).
Version 1.0 April 2026
  • New Leq statistics — rolling energy-average level over 1-, 10-, and 30-minute windows. Leq is the same formula used to compute your dose, applied to a sliding window instead of the whole day. It shows how loud you have actually been — not just right now.
  • New Ln percentile statistics — L10, L50, and L90 computed live over the last 30 minutes. L10 captures your loudest moments (exceeded 10% of the time), L50 is your typical working level, L90 reveals the background noise floor. All three values are shown in the menu bar panel.
  • Impr Safety tab in Settings now explains Leq and Ln with the formula, standard references, and a plain-language description of what each number means in practice.
  • Impr Session average SPL is now computed as a true Leq (energy average), consistent with the rolling Leq statistics and the NIOSH dose formula.
Version 0.9.1 April 2026
  • New Calibration profiles — save the current frequency correction and level offset as a named profile and recall it any time from Settings → Audio.
  • Impr Profiles store the input device name and date for easy identification. Applying a profile restores both the frequency correction and the level offset.
  • Fix Automatic updates via Sparkle now work correctly. Due to a code signing issue in 0.9, this update must be installed manually — future updates will be delivered automatically.
