Lyra
Apollo keyboard volume control for macOS
Version 1.5 May 2026 Latest
  • New App Intents bundle - eleven intents exposed to Shortcuts.app, Spotlight, and Siri. Read state (Get Lyra State, Get Current Monitor Level, Get Current Reference Level), toggle the Apollo (Toggle Mute, Toggle DIM, Toggle Mono), adjust or set the monitor level, and drive Reference Levels (Switch / Cycle / Save). Every intent returns structured values for chaining in Shortcuts. Requires macOS 13+; macOS 12 launches identically without the Shortcuts surface.
  • New App Shortcuts - the most-reached-for intents (Toggle Mute, Switch Reference Level, Set Monitor Level, Get Lyra State, and friends) surface straight in Spotlight without building a Shortcut first.
  • New Focus Filter - Lyra appears in System Settings → Focus → Focus Filters. Attach a "Switch to Reference Level" filter to any Focus and Lyra jumps to that slot when the Focus turns on, with an optional "Restore previous level when Focus ends" toggle.
  • New URL scheme read endpoints - lyra://volume/get, lyra://reference/current, and lyra://state let Shortcuts, Stream Deck, Raycast, and AppleScript pull Lyra's state back. All three require an x-success callback to deliver the data.
  • New x-callback-url chaining - every URL endpoint now accepts optional x-success and x-error parameters. Returned values are appended to the callback as flat query items - no JSON parsing required in Shortcuts.
  • New URL reference page at lyra/api - every endpoint, every return field, with copy-paste examples for Shortcuts, Stream Deck, Keyboard Maestro, and AppleScript.
  • New User Guide menu link - a new User Guide… entry in Lyra's menu opens the user guide in your browser.
  • Fix lyra://volume/set used to return the previous dB value alongside the new tapered position; the response now waits for the Apollo's dB echo so the two readings stay in sync.
  • Fix lyra://reference/current always returns the live tapered and db so you can see where the monitor is between presets, and adds slot=none when no slot matches.
  • Fix Error messages and ranges in URL responses are now plain ASCII (Slot must be 1-3, ?tapered=0.0-1.0) instead of typographic en-dashes that became %E2%80%93 escapes in chained URLs.
Version 1.4 April 2026
  • New Reference Levels - three named monitor presets (default Quiet / Mix / Loud) you can rename and recall. Useful for K-System reference, A/B'ing loudness, or quickly checking a mix at a low level.
  • New Ctrl+F10 / F11 / F12 - jump to Reference Level slot 1, 2, or 3.
  • New Reference Levels submenu - jump to a slot, save the current level into a slot, or rename a slot, all from the menu bar menu.
  • New HUD slot name - when you jump to a Reference Level the on-screen pill shows the slot name (e.g. Mix) so you know exactly where you landed.
  • New URL scheme - drive Lyra from Stream Deck, Keyboard Maestro, Shortcuts, or any other app: lyra://reference/{1|2|3|cycle}, lyra://volume/set?tapered=…, lyra://volume/{up|down} (with ?fine=1), and lyra://{mute|dim|mono}/toggle.
  • New Audita pairing - Audita (our SPL meter, formerly Auris) can now write its calibrations straight into a Lyra slot. After Audita's target-level wizard, click "Send to Lyra…" - Audita ships the calibrated dB SPL to Lyra via the new lyra://reference/{n}/save URL, and the slot starts displaying as e.g. "Mix - 85 dB SPL" in the menu and HUD. Requires Audita 1.0 or later.
Version 1.3 April 2026
  • New Fine-step control - hold Shift while pressing F11/F12 for a half-size volume step.
  • New DIM shortcut - Shift+F10 toggles Apollo's hardware DIM. Attenuation amount is set in UA Console; Lyra just flips the switch.
  • New Mono shortcut - Opt+F10 toggles L+R sum to mono for quick compatibility checks.
  • New Menu items - DIM and Mono appear in the menu bar menu with live checkmarks.
  • Impr Menu bar icon - horizontal bar through the dot when DIM is active; vertical bar when Mono is active. Both can stack.
  • Impr Volume HUD - pill turns amber when DIM is active so the state is obvious at a glance.
