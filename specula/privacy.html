Privacy Policy

Last updated: May 2026

Overview

Specula is a local macOS application. It does not transmit, upload, or share any audio data, file content, or analysis results. All processing happens on-device. There is no telemetry and no analytics.

Audio file access

Specula opens audio files you select via the standard macOS file picker (or drag-and-drop). The entire file is loaded into RAM for analysis and discarded when the app is closed or another file is loaded. Audio is never written elsewhere on disk except when you explicitly export - for example, saving an edited file, exporting a diff WAV, or generating a PDF/JSON report - and even then it goes only to the location you choose.

Speech-detection model

Specula's speech detection runs entirely on your Mac, using the open-source Silero VAD model via the FluidAudio library. No audio, and no information about your files, is ever sent anywhere.

Preferences and settings

Your settings - Layout defaults, Spectrogram colormap and dB range, FFT window, Report toggles - are stored locally on your Mac (in standard macOS preferences) and are never transmitted.

Auto-update checks

Specula uses Sparkle for automatic updates. Sparkle periodically fetches the appcast feed at https://headroomstudio.dev/specula/appcast.xml to check for new versions. This request includes your macOS version and CPU architecture so Sparkle can determine compatibility, but no personal data is transmitted. You can disable automatic checks in the app's menu.

Purchases

When you purchase Specula, payment is processed by an external payment provider. That provider's privacy policy applies to the transaction. Headroom does not receive or store your payment details.

Contact

If you have any questions about this privacy policy, you can reach us at hello@headroomstudio.dev.
